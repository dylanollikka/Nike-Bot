# Nike-Bot

# Purpose

 This project is used to automatically checkout and pay for limited items on the Nike SNKRS website. 
 
 The goal is to checkout in less than 5 seconds
 
# Requirements

  - Python3
  - Selenium

# Usage

Run:

user@desktop$ python3 snkrsbot.py
